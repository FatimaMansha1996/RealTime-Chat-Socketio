Clone the repository:

1)git clone https://github.com/FatimaMansha1996/RealTime-Chat-Socketio.git then
2)cd RealTime-Chat-Socketio



Backup Setup Instructions:


1)cd server
2)npm install
3)node server.js
4)Server will run on: http://localhost:5000


Frontend Setup Instruction:
1)cd client
2)npm install
3)npm run dev
4)Client will run on: http://localhost:5173



Open multiple tabs or devices to test real-time messaging
